1. 该版本对所有不规范的 API 命名进行了统一修复，提升代码规范度和易用性。以下仅为变更示例，请至[类型详情](Linux-keytype)页查看所用类型的具体改动。

    | 变更对象 | 变更详情 | 旧版示例 | 新版示例 |
