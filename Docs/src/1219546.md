> 本文档 API 接口为最新版本接口，后续相关功能的新增都会在此更新，推荐使用最新版本接口。旧版接口文档请参考[历史版本](https://www.volcengine.com/docs/6348/1163751)。

在实时音视频通话场景中，你可以通过调用此接口获取特定应用在指定时间段内的通话时长数据。数据以天为单位进行汇总。

> 注意：由于汇总粒度不同，此处获取的数据和账单通话时长数据差异。最终数据以账单数据为准。
## 注意事项
- 最大请求次数：2次/秒
	
- 最远可查询时间：30 天
	
- 数据粒度：1 天
	
- 数据延迟：8 小时
## 请求说明
- 请求方式：**POST**
- 请求地址：**https://rtc.volcengineapi.com?Action=ListUsages&Version=2023-11-01**
