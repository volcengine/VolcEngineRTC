## 体验说明

1. 下载安装体验包
2. 登录-选择场景模块-【在线KTV】进入
3. 点击【创建房间】开启K歌房或进入某个已存在的K歌房
4. 上麦者可点击【点歌】唤起点歌台进行点歌
5. 点击【调音】，进入调音台，调节 音量、耳返、混响效果
6. 房主可以邀请观众、封锁麦位、禁言嘉宾、踢人下麦等，保证主唱有良好的演唱体验
