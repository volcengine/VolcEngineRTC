## 体验说明

由于音视频场景体验 APP 为双人场景，建议准备两个设备，以获取最佳体验：
1. 下载安装体验包
2. 登录 -> 选择场景模块 ->【音视频通话】进入。
3. 点击右上角美颜按钮，设置美颜特效。
4. 输入用户 ID -> 点击搜索-> 点击呼叫-> 选择视频呼叫/音频呼叫。
