> 本文档 API 接口为最新版本接口，后续相关功能的新增都会在此更新，推荐使用最新版本接口。旧版接口文档请参考[历史版本](https://www.volcengine.com/docs/6348/1163751)。

在实时音视频通话场景中，若需要获取指定房间内的实时用户列表，你可以通过调用此接口实现。

通过指定应用标识和房间 ID，你可以查询到该房间中存在的用户总数以及不可见用户总数。此外，接口还返回可见用户和不可见用户的具体列表，以及房间是否存在的状态信息。
## 注意事项
请求频率：QPS 不得超过 **40**。
## 请求说明
- 请求方式：**GET**
- 请求地址：**https://rtc.volcengineapi.com?Action=GetRoomOnlineUsers&Version=2023-11-01**
