```java
    int state,
    int reason)
```
当白板录像播放状态发生改变时触发此回调。

**传入参数**
| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| state | **int** | 当前状态码<br/>• 0：初始化<br/>• 1：暂停播放<br/>• 2：开始播放<br/>• 3：停止状态 |
| reason | **int** | 状态切换原因代码<br/>• 0：用户或系统主动行为<br/>• 101：开始播放时资源加载<br/>• 102：seek 行为做数据准备<br/>• 601：播放结束<br/>• 602：数据为空<br/>• 603：数据错误<br/>• 604：数据下载失败 |
