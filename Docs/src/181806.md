## 体验说明

1. 打开抖音，扫描二维码，打开小游戏；
	
2. 房主创建一个房间，并进入房间；
	
3. 其他游戏方，加入同一个房间；
	
4. 游戏未开始之前，成员之间使用「按住说话」，进行语音沟通；
	
5. 3个成员全部点击「准备」之后，房主开始游戏；
	
6. 游戏开始
	1. 天黑请闭眼，狼人开始交流本局想法，平民听不到狼人之间的交流声音；
		
	2. 天亮了，每个成员逐一发言，交流不同角色对本轮游戏的看法；
		
	3. 1分钟左右，房间解散，本轮体验结束。
